# Web-Multimedia-Application
<div class="project-description">
  <p>Реализовать веб-мультимедиа проект - «Галерея изображений»</p>

  <div class="functionality">
    <h3>Функционал:</h3>
    <ul>
      <li>Ручное/автоматическое переключение изображений;</li>
      <li>Каждое изображение сопровождается текстовым комментарием;</li>
      <li>Возможность установки времени переключения пользователем (timing);</li>
      <li>Включение/отключение звуковых дорожек (soundtracks);</li>
      <li>Выбор звуковой дорожки (min 3) (или загрузка аудио-файла)</li>
      <li>Реализовать аудио-визуализацию звуковой дорожки (изменение свойств объектов на основе данных аудио потока).</li>
    </ul>
  </div>

  <p>Реализация средствами HTML5, CSS3, JS, SVG, HTML5 Audio API. Без использования сторонних библиотек.</p>
</div>
